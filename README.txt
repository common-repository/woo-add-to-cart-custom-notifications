Set and show custom add-to-cart notifications.

== Description ==

Set custom add-to-cart notifications.
Show custom notification when an product has been added to cart.

== Installation ==


1. Upload the plugin files to the '/wp-content/plugins/woo-add-to-cart-custom-notifications' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

Yes, this is WooCommerce addon, so WooCommerce is required.
